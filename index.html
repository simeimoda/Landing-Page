Sobre mim

Estudante de Análise e Desenvolvimento de Sistemas, dedicado ao aprofundamento técnico no ecossistema JavaScript. Atualmente, foco meus estudos na integração de tecnologias front-end e back-end para construir aplicações escaláveis e funcionais, com boa base em lógica de programação, arquitetura de software e modelagem de sistemas. Tenho me especializado no uso de React para o desenvolvimento de interfaces de usuário modulares e eficientes, buscando aplicar essa tecnologia na criação de experiências web completas.

PrimeFlix

Um catálogo dinâmico de filmes criado para consolidação técnica. Esta aplicação se conecta em tempo real à API oficial do The Movie Database (TMDB) para listar os principais títulos atualmente em exibição no cinema, apresentando avaliações precisas, detalhes e permitindo que o usuário crie e ordene sua própria coleção sob demanda.

LocalStorage & Vanilla JS

Shopping-List-App

Um utilitários intuitivo desenvolvido de forma minimalista para agilizar o planejamento doméstico. O aplicativo permite o controle total e reativo de uma lista dinâmica de supermercado, salvando as informações no navegador por meio de persistência via localStorage para garantir que os dados permaneçam salvos mesmo ao fechar e abrir a aba.
